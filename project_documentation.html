Page-by-Page Features

Detailed functionality of each page in the system

Citizen Portal Features

Login Page

User authentication with username/password, error handling for invalid credentials, redirect based on user role (Citizen/MP/Admin).

Registration Page

New citizen registration with validation for username, email, password strength (min 8 chars), password match confirmation.

Citizen Dashboard

Display statistics: total complaints, pending, in-progress, resolved. Show recent complaints, latest parliamentary speeches with AI summaries.

Submit Complaint

Form with title, description, category selection. Validation for required fields. Success confirmation and redirect to dashboard.

My Complaints

Table view of all submitted complaints with status badges (Pending/In Progress/Resolved), sorting by date, search functionality.

Complaint Detail

Full complaint view with description, category, status timeline, MP reply with AI sentiment analysis, date tracking.

Parliament Speeches

List of all MP speeches with AI-generated summaries, topic tags, sentiment scores, filter by MP or date, search.

Profile Page

View and edit profile: name, email. Display stats: total complaints submitted, resolved count, account info.

MP Dashboard Features

MP Dashboard

Statistics overview: total complaints, pending, resolved. Recent complaints table, quick actions, recent speeches list.

View Complaints

Complete list of all citizen complaints across all constituencies. Filter by status, category, date. Sort functionality.

Reply Complaint

Response form with reply text, status update dropdown. AI suggestion for response based on sentiment analysis.

Add Speech

Upload parliamentary speech: title, full text, summary. AI automatically generates summary, detects topics, analyzes sentiment.

My Speeches

Manage uploaded speeches: view, edit, delete. Display performance metrics (views, engagement).

Admin Panel Features

Admin Dashboard

System-wide statistics: total users, citizens, MPs, complaints, speeches. Charts and graphs for analytics.

Manage Users

View all users, filter by role, activate/deactivate accounts, delete users, view user activity log.

Manage MPs

Add new MP accounts, assign constituencies, edit MP profiles, view MP performance metrics.

System Settings

Configure AI model parameters, set language preferences, manage system preferences, view logs.

Database Schema

Complete database models and relationships

User (Custom User Model)
id (PK) username email password first_name last_name role (citizen/mp/admin) is_active date_joined
Complaint
id (PK) citizen (FK → User) title description category status mp_reply created_at
ParliamentSpeech
id (PK) mp (FK → User) title speech_text summary (AI) date sentiment (AI) topics (AI)
(Future: Translation)
id (PK) speech (FK) language translated_text translated_summary
